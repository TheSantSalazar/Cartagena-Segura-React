# Cartagena Segura - Frontend

Frontend React para la plataforma Cartagena Segura.

## Requisitos

* Node.js 18+
* npm

## Instalación

* git clone https://github.com/TheSantSalazar/Cartagena-Segura-React.git
* Abrir la carpeta con tu editor de codigo o IDE favorito

## Instalar dependencias

* npm install

## Ejecución

* npm run dev
* La aplicación estará disponible en: http://localhost:3000
